# Project Overview
A personal link in bio page built with Python and Flask, where users can add URLs, and the app displays them as styled link cards.

# Tech Stack
Python, Flask, HTML

# File Structure
- app.py: main application
- templates/: HTML files
- static/: CSS files

# Coding Conventions
Use clear variable names, add comments for any logic that isn't obvious, and keep functions small and focused.

# Testing
After making changes, run `python app.py` and confirm the app starts without errors.
